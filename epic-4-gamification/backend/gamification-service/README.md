# Gamification Service

Spring Boot microservice for gamification engine.

## Planned Features
- Points system (earn/spend/transfer)
- Badges and achievements
- Leaderboards
- Challenges and rewards
- Kafka consumer for mission/team events

## Status
> Planned — not yet implemented.

## Tech Stack
- Spring Boot 3.x
- PostgreSQL
- Eureka service discovery
- Kafka event bus
