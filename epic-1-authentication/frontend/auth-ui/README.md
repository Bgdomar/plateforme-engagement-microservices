# Auth UI

Angular micro-frontend for authentication features.

## Pages
- Login (email/password)
- Register (multi-step form)
- Face Authentication (MediaPipe + facial recognition)
- Profile management

## Status
> Currently served by the shared Angular frontend at `shared/frontend/`.
> Pages: `login`, `register`, `face-auth`, `profile`

## Development
```bash
cd shared/frontend
npm install
ng serve
```
